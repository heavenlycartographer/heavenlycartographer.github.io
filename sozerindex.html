<h1 class="title">
How to install latest Intel graphics drivers on Windows 10 Technical Preview
</h1>
</div>
<div class="col-md-2">
</div>
</header>
<section class="col-xs-12">
<p><strong>UPDATE</strong>: Intel released official drivers for Windows 10, you can download them <a href="https://downloadcenter.intel.com/download/25150/Intel-Iris-Iris-Pro-and-HD-Graphics-Production-Driver-for-Windows-10-64-bit">here</a>. After downloading, all you need to do is extract the zip file, go to Device Manager, and select Intel HD graphics and open itâs properties. Select Driver tab, and click on Update Driver &gt; Let me pick from a list of device drivers on my computer &gt; Have Disk and select the inf file.</p>
<p>â</p>
<p>If you would like to install latest Intel drivers out of box to Windows 10 TP, you are out of luck since Intel is blocking the installation of them. With a little tinkering, you can install these drivers to enable more features and control for devices such as Surface Pro 3.</p>
<p>First, you need to download the latest drivers from Intelâs website at <a href="http://downloadcenter.intel.com/">http://downloadcenter.intel.com/</a></p>
<p>If you are using a Surface Pro 3 like me, select Graphics Drivers &gt; Laptop Graphics Drivers &gt; 4th gen Intel with HD 4400 (otherwise select whatever that suits your configuration)</p>
<p><a href="http://i.imgur.com/Dl5bK5w.jpg"><img src="http://i.imgur.com/Dl5bK5w.jpg" alt="" title=""></a></p>
<p>In the next page, select the architecture and then make sure to download the zip package.</p>
<p><img src="http://i.imgur.com/785npcJ.jpg" alt="" title=""> <a href="http://i.imgur.com/XqjLrbf.jpg"><img src="http://i.imgur.com/XqjLrbf.jpg" alt="" title=""></a></p>
<p>Download and extract the zip file, and navigate to Graphics folder, and find the inf file (in this case, itâs igdlh64.inf but it might be different in your case)</p>
<p><a href="http://i.imgur.com/Uzecfti.jpg"><img src="http://i.imgur.com/Uzecfti.jpg" alt="" title=""></a></p>
<p>Open the file and scroll down to section that says âWindows 8.1 Installâ and then change âNTamd64.6.3â to âNTamd64.6.<strong>4</strong>â. This will enable drivers to be installed to Windows 10 âThresholdâ.</p>
<p><a href="http://i.imgur.com/kDwsDPS.jpg"><img src="http://i.imgur.com/kDwsDPS.jpg" alt="" title=""></a></p>
<p>Delete the igdlh.cat file in the same directory.</p>
<p><a href="http://i.imgur.com/RPWXvcq.jpg"><img src="http://i.imgur.com/RPWXvcq.jpg" alt="" title=""></a></p>
<p>Now that you altered this file, this broke the signature of the drivers, so it can no longer be installed as is, since drivers need to be digitally signed to be installed.</p>
<p>To bypass this check, you need to boot with the integrity check disabled. Go to Settings &gt; Update &amp; Recovery &gt; Advanced Startup and click restart now. Follow the instructions there to change startup behavior and disable integrity check (F7).</p>
<p><a href="http://i.imgur.com/mAddyJK.jpg"><img src="http://i.imgur.com/mAddyJK.jpg" alt="" title=""></a></p>
<p>This will restart the computer with integrity check disabled once, so you donât need to change this back. Once itâs back up, go to Device Manager, and select Intel HD graphics and open itâs properties. Select Driver tab, and click on Update Driver &gt; Let me pick from a list of device drivers on my computer &gt; Have Disk and select the inf file you modified before.</p>
<p><img src="http://i.imgur.com/Ik3xTUA.jpg" alt="" title=""> <a href="http://i.imgur.com/AuN6Sy3.jpg"><img src="http://i.imgur.com/AuN6Sy3.jpg" alt="" title=""></a> <a href="http://i.imgur.com/9c12ep8.jpg"><img src="http://i.imgur.com/9c12ep8.jpg" alt="" title=""></a> <a href="http://i.imgur.com/4xbsz7n.jpg"><img src="http://i.imgur.com/4xbsz7n.jpg" alt="" title=""></a></p>
<p>This will start installing the driver, make sure to select install this driver anyway when prompted.</p>
<p><a href="http://i.imgur.com/cdPeiCq.jpg"><img src="http://i.imgur.com/cdPeiCq.jpg" alt="" title=""></a></p>
<p>Now that we installed the driver, everything should be done, right? Unfortunately not. Windows Update will automatically find that there is a driver available (even though you have the latest one), and will try to install that one, overwriting the one you just installed. In previous versions of Windows, you could hide the updates so it wonât install those, however, this functionality isnât there as of Windows 10 TP (build 9926).</p>
<p><del>To enable this functionality, you need to edit the registry. Change HKEY<em>LOCAL</em>MACHINE/SOFTWARE/Microsoft/WindowsUpdate/UX/IsConvergedUpdateStackEnabled to 0, and restart. This will enable the old style Control Panel &gt; Windows Update menu where you can hide updates. Make sure to hide the Intel drivers from there.</del></p>
<p><a href="http://i.imgur.com/9PZgGI5.png"><img src="http://i.imgur.com/9PZgGI5.png" alt="" title=""></a></p>
<p><strong>Update</strong>: In latest builds, ability to hide drivers through Windows Update is not available anymore. You can block installation through System Properties &gt; Hardware &gt; Device Installation Settings.</p>
<p><a href="http://i.imgur.com/i8VsQgV.png"><img src="http://i.imgur.com/i8VsQgV.png" alt="" title=""></a></p>
<p>Or you can try this script: <a href="https://answers.microsoft.com/en-us/insider/forum/insider_wintp-insider_update/how-to-block-automatic-reinstall-of-a-driver-from/bff07224-4ce2-4b42-ab29-68c3a305d730">https://answers.microsoft.com/en-us/insider/forum/insider<em>wintp-insider</em>update/how-to-block-automatic-reinstall-of-a-driver-from/bff07224-4ce2-4b42-ab29-68c3a305d730</a></p>
<p>This allows you to access Intel Graphics Control Panel, where you can tweak various settings.</p>
<p><a href="http://i.imgur.com/c20QVmX.jpg"><img src="http://i.imgur.com/c20QVmX.jpg" alt="" title=""></a></p>
